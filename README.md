## 🔥 Key Features

### 🏢 Professional Examination Portal
- **Subject Hierarchy**: Structured paths (e.g., MID Examination > CSNE > DSNM) to replicate real academic categories.
- **Instructional Dashboard**: Detailed exam guidelines and a professional SLIIT-themed interface.
- **Data Privacy Assurance**: A transparent disclaimer ensuring no personal data or examination results are collected.

### ⏲️ Realistic Quiz Workflow
- **Precision Timer**: A 1-hour (3600s) countdown timer starting exactly upon quiz initiation.
- **Auto-Submission**: Automated quiz closing and submission when the time expires to maintain academic integrity.
- **Interactive Navigation**: A sidebar grid for easy monitoring of answered and un-answered questions.
- **Instant Feedback**: Comprehensive result summary with score percentage and per-question correctness.

### 💻 Security & Accessibility
- **Desktop Restriction**: A dedicated overlay blocks mobile access, ensuring the platform is used only in a secure desktop/laptop environment.
- **Enquiry System**: Professional, themes-consistent "Inquiry" channel for student support.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.x or later)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/VehanRajintha/Netexam_clone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Netexam
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build
Generate an optimized production build:
```bash
npm run build
```
